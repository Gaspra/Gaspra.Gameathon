Join The JustGiving Team

Want to set up a JustGiving page to go alongside the Gameathon marathon?

You can donate and encourage others to donate through the main page: 15below-gameathon2019 or set up your own JustGiving page and join the 15below team much like Alan has. In order to set up your own page just follow the instructions below:


1. Go to https://www.justgiving.com and create an account if you don’t have one already
2. When logged in, drop down the menu to the right of your profile pic and select “Your Fundraising”
3. Make a new page:
- Raising money for A Charity
- Search on “Brighton housing trust” and select it from the top of the Charities section
- Select “Personal Challenge”
- Select “a personal streaming / gaming event” and fill out the rest of the fields on the page, then create the page

4. Now go to the team page: https://www.justgiving.com/teams/15below-Gameathon2019
5. “Join the team”
6. Select the new page you have created and click “Join Team”
7. Once the admin has given the OK you will get a “congratulations” email, and you are in.
8. Personalise your page and send the link to everyone and their dog.
9. Have lots of gaming fun on 18th October

The Gameathon takes place at 6:00pm (BST) on the 18/10/19 get involved, join the Slack channel, donate through JustGiving or watch the event on Twitch.tv!
